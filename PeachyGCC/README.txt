PeachyGCC is a GameCube controller emulator for GBA that allows you
to play any GCC compatible game with just your GBA and the GCN link cable.

Comes with many features not found on the original controller
-Input display
-Input macros
-Modifier keys
-Turbo buttons
-Auto accel
